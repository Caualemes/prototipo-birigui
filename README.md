# Protótipo Birigui IG White Label

Protótipo desenvolvido como parte da disciplina **Extensão III**, ministrada pela professora **Helen**, no **Instituto Federal de São Paulo - IFSP**.

O projeto apresenta uma proposta de portal white-label voltado à divulgação da Indicação de Procedência do Calçado Infantil de Birigui, reunindo páginas de empresas, eventos, roteiros, galeria, contato e uma área administrativa simulada.

Atuei como **líder do grupo de design**, contribuindo na definição visual, estrutura das páginas, organização dos módulos e experiência de navegação do protótipo.

## Tecnologias utilizadas

- React
- Vite
- Tailwind CSS
- Lucide React
- Framer Motion
